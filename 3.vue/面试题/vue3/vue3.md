### 虚拟 DOM 重写

基于 Proxy 的数据响应式
Vue 2 的响应式系统使用 Object.defineProperty 的 getter 和 setter。Vue 3 将使用 ES2015 Proxy 作为
其观察机制，这将会带来如下变化：
组件实例初始化的速度提高 100％
使用 Proxy 节省以前一半的内存开销，加快速度，但是存在低浏览器版本的不兼容
为了继续支持 IE11，Vue 3 将发布一个支持旧观察者机制和新 Proxy 版本的构建
